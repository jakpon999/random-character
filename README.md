# Random Character

Aplikacja na androida z przypiętym API. Pokazuje losową postać z serialu Rick and Morty wraz z jej danymi.

## Funkcjonalności

- Pokazywanie losowej postaci z serialu Rick and Morty wraz z jej danymi 
- Przycisk „reload” losujący następną postać z bazy danych

## Technologie

- Java
- Android Studio
